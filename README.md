Description:
- This program implements the highest value longest common sequence (HVLCS). It computes a common subsequence of A and B that maximizes the total value, and outputs both this maximum value and the corresponding subsequence.

Instructions to compile and run the code:
- Clone the repository
- Navigate to the project root directory 
- In src/main.py, set the filename inside open() to the wanted input file
- Run the program using python src/main.py
    - If your system uses "py", run: py src/main.py
    - If your system uses "python3", run: python3 src/main.py

Assumptions:

Input
- The first number in the input file is K, the number of characters in the alphabet
- Rows afterward should follow this format:
    - x1 v1
    - x2 v2
    - . . .
    - xK vK
- The 2nd to last row is A, the first string
- The last row is B, the second string

Output
- The first number in the output file is the maximum value found
- The second row is the corresponding subsequence
